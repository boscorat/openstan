# About

The **About** dialog shows version information, links, and licence details for openstan and its dependencies.

---

## Opening the About dialog

Click the **About** button in the top-right corner of the application title bar.

![About dialog](../assets/screenshots/about.png#only-light)
![About dialog](../assets/screenshots/dark/about.png#only-dark)

---

## Contents

| Item | Description |
|---|---|
| **Logo and tagline** | The openstan logo, theme-aware (adapts to light/dark mode). |
| **Version** | The currently installed version of openstan (e.g. `0.1.2`). |
| **openstan.org** | Link to the openstan website. |
| **GitHub repository** | Link to the openstan source code on GitHub. |
| **Powered by bank\_statement\_parser** | The version of the underlying parsing library in use. |
| **bank\_statement\_parser repository** | Link to the BSP source code on GitHub. |
| **Copyright** | © 2025 Jason Farrar |
| **Licence** | Link to the full GPL v3+ licence text. |

All links open in your system's default web browser.
